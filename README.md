# App

تطبيق ويب بسيط (PWA) بيعرض فيديو Full Screen. لما تفتحه أول مرة هتلاقي زرار "اضافة فيديو"، تدوس عليه، الجاليري بتاعك هيفتح، تختار الفيديو، وهيتشغل على كامل الشاشة. الفيديو بيتخزن جوه الجهاز نفسه، فلما تفتح التطبيق تاني الفيديو هيفضل موجود من غير ما تختاره تاني. لو حبيت تغيره، اضغط على الفيديو ثانية وحدة وهيظهرلك الزرار تاني.

## 1) غيّر اسم التطبيق واللوجو

### الاسم
في ملف `manifest.json` غيّر القيم دي:
```json
"name": "اسم التطبيق",
"short_name": "اسم قصير",
"description": "وصف بسيط"
```

كمان في `index.html` غيّر:
```html
<title>App</title>
```

### اللوجو
حط صورتين مكان الملفين اللي موجودين في مجلد `icons/`:
- `icons/icon-192.png` (مقاس 192×192 بكسل)
- `icons/icon-512.png` (مقاس 512×512 بكسل)

خليهم بنفس الاسمين بالظبط عشان الكود يلاقيهم من غير ما تغيّر أي حاجة تانية.

## 2) رفع المشروع على GitHub Pages

1. اعمل Repository جديد على GitHub (Public).
2. ارفع كل ملفات المشروع دي بالظبط زي ما هي (نفس الأسماء والمجلدات):
   - `index.html`
   - `style.css`
   - `script.js`
   - `manifest.json`
   - `service-worker.js`
   - مجلد `icons/` بالصورتين اللي جواه
3. من إعدادات الـ Repository روح على تبويب **Settings > Pages**.
4. تحت **Source** اختار **Deploy from a branch**، وبعدين اختار البرانش `main` والمجلد `/ (root)`.
5. احفظ، واستنى دقيقة أو اتنين، هيديك لينك زي:
   ```
   https://username.github.io/repo-name/
   ```

## 3) التثبيت على الموبايل

- **أندرويد (كروم):** افتح اللينك، هيظهرلك اقتراح "إضافة إلى الشاشة الرئيسية" أو من قائمة المتصفح (⋮) اختار "Add to Home screen".
- **آيفون (سفاري):** افتح اللينك، دوس على زرار المشاركة (المربع وفيه سهم لفوق)، وبعدين اختار "Add to Home Screen".

بعد كده هيظهر على الشاشة الرئيسية بشكل تطبيق عادي، بالاسم واللوجو اللي حطيتهم، ومن غير أي شريط متصفح.

## ملاحظات

- الفيديو بيتخزن محلي على جهاز المستخدم بس (IndexedDB)، مفيش سيرفر ومفيش رفع لأي مكان.
- لازم اللينك يكون شغال بـ HTTPS عشان الـ Service Worker والتثبيت يشتغلوا (GitHub Pages بيدي HTTPS تلقائي).
